# Utilisation 

## Création environnement virtuel

```bash
make venv
```

## Mise à jour environnemetn virtuel

```bash
make update
```

## Lancement application

Utilisation de flask

```bash
make run_flask
```
--> http://127.0.0.1:5000 


Utilisation de Gunicorn

```bash
make run_gunicorn
```
--> http://127.0.0.1:8000
